### Links and Image Styling

  • list item * list item

  • this is an internal link to nowhere 

    [this is a link to nowhere](#)
  • This is an ***external*** link to nowhere```javascriptThis is an ***external*** link to nowhere```
  • This is an ***external*** link to nowhere that opens in a new tab```This is an ***external*** link to nowhere that opens in a new tab```

  • ![this is a broken imageStyle, which is showing descriptive/alternative text - note the use of "!"](#)

    • GIfs and images must be inserted in the following (HTML) format with a class assignment of "imageStyle" to use uniform styling constraints.

      * GIFs require an additional class assignment of "**shadowing**"
      <img class="image shadowing" alt="SCREENCAST: changing branches / this is a functional gif" src={require('./assets/README_changeBranch.gif').default} />

    • class imageStyle applies:

      .imageStyle {
          margin: 20px 0;
          padding: 10px;
      }

    • class shadowing applies:

      .shadowing {
          box-shadow: 0px 4px 5px 1px rgba(0, 0, 0, .25); 
      }

    • Background shadows are applied in our image editor(s). Shadow dimmensions are as follows:

      Drop Shadow
               Blur:    25(px)
               X:       10(px)
               Y:       10(px)
               Color:   000000
               Opacity: 15%
